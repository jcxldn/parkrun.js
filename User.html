(async) getClubs() → {Promise.<clubsResult>}

Source:

Get the user's Parkrun Clubs (for milestone runs / duties)

Example
const user = .....


await user.getClubs()

// Example Response:

{
  ParkrunClub: { id: 'c3', name: '250+ Club' }
  JuniorClub: { id: 'j0', name: 'No Club' },
  VolunteerClub: { id: 'v1', name: 'Volunteer 25+ Club' }
}
Throws:

  • ParkrunJS Networking Error.

    Type

  • Error when no data is available, usually because of a new account with no runs.
